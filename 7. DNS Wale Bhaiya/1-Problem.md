# DNS Wale Bhaiya

### Problem Link: [Click Here](https://www.hackerrank.com/contests/logic-loot/challenges/dns-wale-bhaiya)

## Problem Statement

Students are waiting to get printouts of their assignments. The number of pages each student wants to print is given in an array. DNS wale bhaiya takes **5 seconds** to print each page.

Write a program to calculate the total time (in seconds) required for DNS wale bhaiya to print all the pages for all students.

## Input Format

- The first line contains an integer `N` — the number of students.
- The second line contains `N` integers, where each integer represents the number of pages a student wants to print.

## Constraints
```
1 ≤ N ≤ 10⁵
1 ≤ pages[i] ≤ 1000
```

## Output Format

Print the total time (in seconds) taken by DNS wale bhaiya to print all the pages.

## Sample Input 0
```
3
4 2 6
```

## Sample Output 0
```
60
```

## Sample Input 1
```
5
1 1 1 1 1
```

## Sample Output 1
```
25
```
