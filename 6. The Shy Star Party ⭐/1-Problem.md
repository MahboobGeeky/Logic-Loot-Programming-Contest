# The Shy Star Party ⭐

### Problem Link: [Click Here](https://www.hackerrank.com/contests/logic-loot/challenges/the-shy-star-party)

## Problem Statement

Every year, stars in the galaxy organize a Star Party. But the stars are very shy 🫣. They don't want to stand in a straight line, so they form a triangle shape where each row has more stars than the previous row.

Your job as the official galaxy programmer is to print the star formation.

Given a number **N**, print a triangle of stars where:
- The first row has 1 star
- The second row has 2 stars
- The third row has 3 stars
- And so on...

Each row should have stars separated by a space.

## Input Format

A single integer `N` — the number of rows the shy stars want to form.

## Constraints
```
1 ≤ N ≤ 50
```

## Output Format

Print the triangle pattern of stars.

## Sample Input 0
```
3
```

## Sample Output 0
```
*
* *
* * *
```
