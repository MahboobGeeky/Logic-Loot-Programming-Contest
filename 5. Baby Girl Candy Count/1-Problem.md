# Baby Girl Candy Count

### Problem Link: [Click Here](https://www.hackerrank.com/contests/logic-loot/challenges/baby-girl-candy-count)

## Problem Statement

A baby girl named Lily loves candies. Her parents give her candies every day.
However, Lily only likes **pink** candies. She ignores all other colors.

You are given a list of candies with their colors. Your task is to count how many pink candies Lily will eat.

## Input Format

- The first line contains an integer `n` — the total number of candies.
- The second line contains `n` space-separated strings representing the colors of the candies.

## Constraints
```
1 ≤ n ≤ 100
Candy colors are lowercase strings.
```

## Output Format

Print a single integer — the number of pink candies.

## Sample Input 0
```
3
pink pink pink
```

## Sample Output 0
```
3
```

## Sample Input 1
```
4
blue red green yellow
```

## Sample Output 1
```
0
```

## Sample Input 2
```
6
pink blue blue pink pink red
```

## Sample Output 2
```
3
```
