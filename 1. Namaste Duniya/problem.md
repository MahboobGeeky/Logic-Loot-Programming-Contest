# Namaste Duniya

## Problem Statement

Write a program to print `Namaste Duniya`.

---

## Input Format

There is no input required for this problem.

---

## Constraints

- The output must exactly match `Namaste Duniya`
- The output is **case-sensitive**
- There must be a single space between the two words

---

## Output Format

Print the string:
```
Namaste Duniya
```

---

## Sample Output
```
Namaste Duniya
```

---

## Solution

### Python
```python
print("Namaste Duniya")
```

### C
```c
#include <stdio.h>
int main() {
    printf("Namaste Duniya\n");
    return 0;
}
```

### C++
```cpp
#include <iostream>
using namespace std;
int main() {
    cout << "Namaste Duniya" << endl;
    return 0;
}
```

### Java
```java
public class Solution {
    public static void main(String[] args) {
        System.out.println("Namaste Duniya");
    }
}
```

### JavaScript
```javascript
console.log("Namaste Duniya");
```
