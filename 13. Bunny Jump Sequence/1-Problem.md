# Bunny Jump Sequence

### Problem Link: [Click Here](https://www.hackerrank.com/contests/logic-loot/challenges/bunny-jump-sequence)

## Problem Statement

A bunny is playing a jumping game in the park.

The bunny follows a special jumping pattern:
- The first jump is **0** meters.
- The second jump is **1** meter.
- Every next jump is the **sum of the previous two jumps**.

This creates a sequence like:
```
0 1 1 2 3 5 8 13 ...
```

Given an integer `n`, print the **nth jump distance** of the bunny.

## Input Format

A single integer `n`.

## Constraints
```
0 ≤ n ≤ 40
```

## Output Format

Print the nth number.

## Sample Input 0
```
2
```

## Sample Output 0
```
1
```

## Sample Input 1
```
3
```

## Sample Output 1
```
2
```

## Sample Input 2
```
4
```

## Sample Output 2
```
3
```
