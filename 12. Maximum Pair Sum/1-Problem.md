# Maximum Pair Sum

### Problem Link: [Click Here](https://www.hackerrank.com/contests/logic-loot/challenges/maximum-pair-sum-1)

## Problem Statement

You are given an array of **N** integers. Your task is to find the maximum possible sum of any two different elements in the array.

Return the maximum sum.

## Input Format

- First line contains an integer `N` — number of elements.
- Second line contains `N` space-separated integers.

## Constraints
```
2 ≤ N ≤ 1000
-10⁵ ≤ Ai ≤ 10⁵
```

## Output Format

Print a single integer — the maximum sum of two different elements.

## Sample Input 0
```
4
1 2 3 4
```

## Sample Output 0
```
7
```

## Sample Input 1
```
3
10 5 8
```

## Sample Output 1
```
18
```

## Sample Input 2
```
6
-1 -2 -3 -4 -5 -6
```

## Sample Output 2
```
-3
```
