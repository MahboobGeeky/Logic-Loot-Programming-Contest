# Last Bench Safety Check

### Problem Link [Click Here](https://www.hackerrank.com/contests/logic-loot/challenges/last-bench-attendance-trick)

## Problem Statement

A student sits on the last bench and wants to know if he is safe from attendance checking.

The teacher checks attendance **only if** the number of students in class **N** is greater than **40**.

Print:
- `Attendance Checked` if N > 40
- `Safe Today` otherwise.

## Input Format

A single integer **N**.

## Constraints
```
1 ≤ N ≤ 200
```

## Output Format

Print:
- `Attendance Checked` if N > 40
- `Safe Today` otherwise.

## Sample Input 0
```
35
```

## Sample Output 0
```
Safe Today
```

## Sample Input 1
```
41
```

## Sample Output 1
```
Attendance Checked
```
