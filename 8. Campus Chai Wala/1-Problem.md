# Campus Chai Wala

### Problem Link: [Click Here](https://www.example.com)

## Problem Statement

In campus chai wala, students come one by one to buy tea.
Each student buys some cups of chai.

At the end of the day, the chai wala wants to know which student bought the maximum number of chai cups.

Given the number of chai cups bought by each student, find the maximum cups bought by any student.

## Input Format

- The first line contains an integer `n` — the number of students.
- The second line contains `n` integers representing the number of chai cups bought by each student.

## Constraints
```
1 ≤ n ≤ 1000
1 ≤ cups ≤ 1000
```

## Output Format

Print a single integer — the maximum number of chai cups bought by a student.

## Sample Input 0
```
1
1
```

## Sample Output 0
```
1
```

## Sample Input 1
```
4
10 20 30 40
```

## Sample Output 1
```
40
```

## Sample Input 2
```
4
6 6 6 6
```

## Sample Output 2
```
6
```
